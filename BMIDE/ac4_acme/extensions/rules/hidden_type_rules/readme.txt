1 file per Business Object main family (Item, Form, Folder, Dataset) and accessor and access type

example : 
Item_hidden_4_all.xml 				(Accessor = Organization, condition = "isTrue")
Item_displayed_4_dba.xml 			(Accessor = dba Group, condition = "isFalse")
Item_displayed_4_Engineering.xml 	(Accessor = Engineering Group, condition = "isFalse")

Dataset_hidden_4_all.xml 			(Accessor = Organization, condition = "isTrue")
Dataset_displayed_4_dba.xml 		(Accessor = dba Group, condition = "isFalse")
Dataset_displayed_4_Engineering.xml (Accessor = Engineering Group, condition = "isFalse")
